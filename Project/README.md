# CIFAR10 CNN
* CNN networks classify 10 classes: 
  * 0: airplane
  * 1: automobile
  * 2: bird
  * 3: cat
  * 4: deer
  * 5: dog
  * 6: frog
  * 7: horse
  * 8: ship
  * 9: truck

* CNN Block:
  * RGB2Gray 
  * Convolution Block:
    * Convol Multi
    * Max pooling
    * Relu
  * Fully-Connected Block:
    * FC node
